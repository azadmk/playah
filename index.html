About this project

Playo is an interactive HTML5 player that focuses in the really efficient new features of HTML5. In order to keep it simple and agile, I designed the whole player with Windows 8 style, the new design language of Microsoft.

Also, it is built with the latest web technologies and practices in mind, such as CSS3 and LESS, the CSS preprocessor.

For a more technical description, head over to Github!

I would be glad to get your feedback via Github issues, Twitter or email.

The songs that are used in this demo are licensed under Creative Commons.
